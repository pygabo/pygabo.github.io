# imgabo.dev

Personal landing page built with [Astro](https://astro.build/).

## Stack

- **Astro 5** - Static site generator
- **TypeScript** - Strict mode
- **GitHub Actions** - CI/CD
- **GitHub Pages** - Hosting

## Pages

- `/` - Landing
- `/colaborar` - How to work with me (ES)
- `/working-with-me` - How to work with me (EN path)
- `/books` - Favorite books

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:4321](http://localhost:4321).

## Build

```bash
npm run build
```

## Deploy

Push to `master`. GitHub Actions handles the rest.
